## 产品说明文档 / 使用手册

### 1. 产品简介

Spell Lab 是一款在线英语单词听写学习网站，帮助你通过“听写—纠错—复习”的方式高效记忆单词。你可以使用系统词书学习，也可以上传自定义词书，随时查看训练记录与学习趋势。

### 2. 快速上手

#### 2.1 注册与登录

首次使用请先注册账号，完成后登录进入系统。登录成功后会看到顶部导航栏，包含：听写、词书、错词本、训练记录、我的。

#### 2.2 顶部导航说明

- 听写：选择词书与章节开始训练
- 词书：浏览系统词书与管理自定义词书
- 错词本：集中复习拼写出错的单词，并进行 AI 训练
- 训练记录：查看学习词数、时长与趋势
- 我的：个人信息、设置与反馈

### 3. 词书管理

#### 3.1 词书类型

- 系统词书：平台内置，开箱即用
- 自定义词书：你自行创建并导入数据

#### 3.2 创建自定义词书

进入“词书”页面，点击“添加自定义词书”，输入名称并确认创建。创建后可立即导入单词文件。

#### 3.3 导入词书文件

支持格式：

- CSV（.csv）
- Excel（.xls / .xlsx）

列格式要求：

- 支持“有表头”或“无表头”
- 表头列名可识别：word / 单词、meaning / 释义、phonetic / 音标 / pronunciation
- 无表头时默认按顺序：第 1 列单词、第 2 列释义、第 3 列音标

注意事项：

- 文件编码建议使用 UTF-8；如是中文环境导出也可使用 GBK
- 若出现“无法读取文件”提示，建议关闭 Excel 后重新选择文件上传

### 4. 听写学习

#### 4.1 选择词书与章节

在“听写”页面选择词书，进入对应章节列表。每个章节卡片会显示单词数、学习进度与正确率。

#### 4.2 进入听写

点击章节的“听写”进入训练模式。系统将朗读单词，你输入拼写后提交结果。

#### 4.3 听写结果

完成训练后会生成成绩页，展示正确率与错词/对词列表，便于复盘。

### 5. 错词本

#### 5.1 错词列表

错词本会自动收集你的错误单词，支持集中复习与筛选操作。

#### 5.2 AI 训练

错词本提供 AI 训练模式，可能包含选择题或填空题，帮助你加深记忆。完成后可查看训练结果与历史记录。

### 6. 训练记录

训练记录页面展示：

- 今日学习与复习词数
- 今日学习时长
- 累计学习词数与时长
- 近期学习趋势图

### 7. 我的

#### 7.1 个人资料

可查看个人信息，并设置头像与封面图。

#### 7.2 听写设置

可调整朗读语速与重复次数，使听写节奏更贴合自身习惯。

#### 7.3 反馈与账号

支持提交反馈；可在此退出登录或注销账号。

### 8. 常见问题

#### 8.1 导入失败/无法读取文件

可能原因：

- 文件正在被 Excel 或同步软件占用
- 文件格式不符合要求

解决建议：

- 关闭 Excel 后重新上传
- 重新保存为 CSV 或 Excel 格式

#### 8.2 听写没有声音

请检查：

- 浏览器是否允许网站播放声音
- 系统音量与浏览器音量是否静音

#### 8.3 训练记录没有更新

请刷新页面或重新登录；若仍未更新，建议联系管理员检查服务状态。

## 本地运行

### 前端

```
cd frontend
npm install
npm run dev
```

### 后端

```
cd backend
mvn spring-boot:run
```
